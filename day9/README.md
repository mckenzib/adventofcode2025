# Day 9: Movie Theater

## Problem Description
Find the largest rectangle that uses two red tiles (given as input coordinates) for two of its opposite corners.
The area is calculated as `(|x1 - x2| + 1) * (|y1 - y2| + 1)`.

## Usage
Run the solution with the input file as an argument:

```bash
python3 solution.py input.txt
```

## Example
Based on the example input:
```
7,1
11,1
11,7
9,7
9,5
2,5
2,3
7,3
```
The output should be `50`.
