# Day 2: North Pole Gift Shop Database Fixer

This solution is a standalone HTML/JavaScript application that visualizes the "deep scan" process for finding invalid gift IDs.

## Problem Description
The system needs to find invalid IDs based on a complex checksum algorithm involving repeated strings. This tool visualizes the search and fixes the database.

## How to Run
Simply open the `northpolegiftshopsolver.html` file in any modern web browser.

1.  Open the file:
    ```bash
    open northpolegiftshopsolver.html
    ```
    (or double-click it in your file explorer)

2.  Paste your puzzle input into the text area.
3.  Click **INITIATE DEEP SCAN**.
